## 🧠 File Parser Engine

PepeluGPT includes a sophisticated file parsing engine with:

### Features:
- ✅ **Multi-format parsing**: `.pdf`, `.docx`, `.html`, `.xml`, `.pptx`, `.txt`, `.xls`, `.xlsx`
- 🔄 **Retry mechanism** for borderline failures
- 🧪 **Validation** with auto-tagged failure types
- 🧬 **Tokenization & normalization** for AI consumption
- 🔍 **Metadata extraction** & language detection
- 📊 **Batch progress tracking** with logging
- 🔒 **Privacy-first design** for offline processing

### Parser Configuration:
```python

    "max_retries": 1,
    "lowercase": True,
    "strip_html": True,

}
```

## 🛠 Troubleshooting

| Issue | Solution |
|-------|----------|
| `langdetect` fails | Ensure text length >20 chars before detection |
| Parsing crashes on `.pdf` | Check PyMuPDF is installed correctly |
| Empty search results | Verify documents are in `cyber_documents/` folder |
| Vector DB loading fails | Run `python pepelugpt.py setup` to rebuild |
pip install pytest black flake8  # Development tools
